# Cấu hình

File: **`app/docs/configuration/page.mdx`** → **`/docs/configuration`**.

## `next.config.mjs`

Wrapper `nextra()` bọc cấu hình Next.js — chỉnh MDX, theme, search trong đó.

## Sidebar

File **`app/docs/_meta.tsx`** điều khiển thứ tự và nhãn trong sidebar của nhánh `/docs`.

Quay lại [Bắt đầu](/docs/getting-started).
