# Gradient Descent

Gradient descent la thuat toan toi uu co ban de huan luyen mo hinh.

1. Khoi tao tham so ngau nhien
2. Tinh gradient cua ham mat mat
3. Cap nhat tham so theo huong nguoc gradient

Cong thuc cap nhat:

`theta = theta - learning_rate * gradient`
